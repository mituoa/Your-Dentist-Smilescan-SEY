# Section Brand Decisions

This file extends the brand system with three new section labels chosen to feel durable, premium, and meaningful.

## Selected labels
- `Portrait`: replaces `Profil`
- `Settings`: keep familiar label for `Einstellungen`
- `Atlas`: proposed label for `Dashboard`

## Why these three

### Portrait
- Best fit for the public doctor and practice presence
- Feels curated, human, and timeless
- Reads like a house label, not a software feature

### Settings
- Keeps the industry-standard wording users already expect
- Reduces friction in admin, access, and configuration areas
- Maintains clarity while the other section labels carry brand character

### Atlas
- Recasts the dashboard as a strategic overview rather than a technical control panel
- Works well for submissions, tasks, and recent activity in one place
- Feels broad, calm, and durable

## Suggested UI wording
- `Portrait`
- `Edit Portrait`
- `Open Portrait`
- `Settings`
- `Open Settings`
- `Atlas`
- `Open Atlas`
