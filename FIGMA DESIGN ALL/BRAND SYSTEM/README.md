# Your Dentist Brand System

This folder contains a neutral umbrella brand for `SCH LTD` plus a consistent logo family for the current product areas.

## Architecture
- `SCH LTD`: legal entity only, not the customer-facing name.
- `Your Dentist`: parent brand and neutral domain anchor.
- `SmileScan`: visual intake, triage, and scan-led case capture.
- `Journals`: public publishing, trust-building, and profile storytelling.
- `Relay`: the proposed new name for the current Tasks area.
- `Portrait`: public doctor and practice presence.
- `Settings`: settings, access, team, and brand control.
- `Atlas`: dashboard and practice overview.

## Why `Relay`
- It describes movement between people, not just a static to-do list.
- It sounds neutral and premium enough to sit next to `SmileScan` and `Journals`.
- It matches the existing workflow model: assign, review, approve, follow through.

## Files
- `brand-showcase.html`: quick local preview of the full system.
- `logos/<brand>/logo-horizontal.svg`: primary lockup.
- `logos/<brand>/logo-mark.svg`: icon-only version.

## Suggested usage
- Use `Your Dentist` on landing pages, public practice pages, and shared ecosystem surfaces.
- Use `SmileScan` for scan capture, submissions, and intake flows.
- Use `Journals` for public articles, thought leadership, and profile content.
- Use `Relay` for team tasks, approvals, and follow-up workflows.
- Use `Portrait` for the public doctor/practice page and its editor.
- Use `Settings` for settings, team, branding, slug, and appearance controls.
- Use `Atlas` for the doctor's overview page, incoming signals, and operational summary.

## Color direction
- `Your Dentist`: evergreen trust.
- `SmileScan`: brighter clinical teal.
- `Journals`: warm editorial copper.
- `Relay`: structured cobalt.
- `Portrait`: quiet merlot.
- `Settings`: slate administration.
- `Atlas`: deep verdigris overview.
